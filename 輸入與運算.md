# 輸入與運算

## 方形面積計算

```c
#include<stdio.h>
int main() {
    float Length, Width, Area;

    printf("長(公分)：");//" "代表字串，' '代表字元
    scanf("%f",&Length);//利用佔位符輸入資料給變數length，遇到確認鍵或空白結束輸入，但不是以%xxx開頭時則遇到確認鍵結束輸入，按下確認鍵後都留換行字元在緩衝區

    printf("寬(公分)：");
    scanf("%f",&Width);

    Area = Length * Width;
    printf("方形面積為：%f平方公分\n",Area);/將變數length輸出給第一個佔位符

    system("Pause");//使用者按下一個鍵繼續執行程式
    return 0;
}
```

## 佔位符

| 佔位符 | 說明                      |
|--------|---------------------------|
| `%d`   | 有號十進制整數            |
| `%hd`  | short整數                  |
| `%ld`  | long整數                   |
| `%u`   | 無號十進制整數             |
| `%o`   | 無號八進制整數（小數點補到第六位） |
| `%x`   | 無號十六進制整數           |
| `%e`   | 科學計數輸出符點數         |
| `%f`   | float浮點數                |
| `%.lf` | double浮點數               |
| `%g`   | 較短寬度輸出符點數         |
| `%a`   | 十六進位科學計數輸出符點數 |
| `%c`   | 字元                        |
| `%s`   | 字元陣列                    |
| `%p`   | 指標                        |

## 格式化字符串

`%[旗標][寬度][.精準度][長度]類型`

- 旗標：
  - `+`: 顯示出正負號。
  - `-`: 靠左對齊。
  - `0`: 前面加0至最小寬度。
  - `#`: 佔位符遇到x、o時，輸出前面會加上0、0x；遇到e、f時，強制輸出小數；遇到g時，強制輸出剩下的0。

- 寬度：
  - 大於輸出長度時加上空格，可用*代替，並在後面增加數字。

- .精準度：
  - 若為輸出符點數則代表小數位數，若為輸出字元則代表字元字數，可用*代替，並在後面增加數字。
 
- 長度：
  - h：短整型
  - l：長整型
 
- ex：

```c
printf("%+5.2f",num);
printf("%.2lf",num2);
printf("%*.*lf",5,2,num2);//第一個數字5是放入*裡
```
  

## 變數、常數各種型態

### 儲存類別

- `auto`: 自動判別宣告類型，因使用極少及多餘。
  - 許多環境已刪掉此用法。

- `register`: 變數儲存在暫存器位址，速度快容量小。
  - 通常用在區域變數。
  - 不可使用一元&運算符 (e.g., `register int x;`).

- `extern`: 在檔案中共享宣告的全域變數或函數。
  - `extern int x;`、`extern int f(int x);`

- `mutable`: 在類別裡讓指定常數變成變數。
  - `mutable int x;`

### 常數修飾詞

- `const`: 將變數變為常數 (無法修改值).

  - `const double pi = 3.14159f;`

### 修飾詞

- 有號數: `signed` (4byte)
  - 包含負值，預設為有號數。

- 無號數: `unsigned` (4byte)
  - 無包含負值，0不是負值。

### 變數型態

#### 整數


- 短整數`short` (2byte)
- 整數`int` (4byte)
- 長整數`long` (4byte)
- `size_t` (4byte)
- `long long` (8byte)
- Example: `1`, `-66`, `0x1b`
  
#### 浮點數

- 單精度 `float` (4byte)
- 雙精度 `double` (8byte)
- 長雙精度 `long double` (8byte)
  - 為有號數
  - 無法被修飾詞改動
- Example: `1.65`, `-13.1`, `3.14159`

#### 布林

- `bool`: 存 0 與 1，分別為 `false`、`true`.
  - 因為只有0、1所以為無號數
  - 無法被修飾詞改動
- Example: `1`, `0`, `true`, `false`

#### 字元

- `char` (1byte)
- `wchar_t` (2byte)
  - 預設為有號數。
  - 編譯、使用時變為無號數。
  - `char` 就是 ASCII 字元，`wchar_t` 儲存 Unicode 碼。
    
    ##### ASCII字元:
    - 英文跟數字與符號<=>十進制與十六進制
    
    ![image](https://github.com/xixa3333/C-Textbook/blob/main/ASCII%E5%AD%97%E5%85%83%E8%A1%A8.gif)
    
    ##### Uniclode字元(萬國碼):
    - 各國語言<=>十六進制
  
  - Example: `'1'`, `'a'`, `'@'`, `65`

#### 空值

- `void`: 經常用於宣告函數。
  
  > 被 `void` 宣告的函數無法回傳。

#### Example Declarations:
  ```c
  int x;
  float a, b;
  int num = 5, num2;
  ```
  
#### 注意事項:
  ```c
  char x = (char)(65); //在C做使用
  char x = char(65);  //在C++做使用
  ```

## 緩衝區

>有512個位元組的大小

1. 執行輸入函數。
2. 緩衝區沒有值時讓使用者輸入至緩衝區。
3. 將緩衝區的值丟入變數。
4. 沒取到的值留在緩衝區。

- `fflush(stdin);` 清除緩衝區

|函數|描述|
|:-----:|:-----:|
|getchar()|只輸入一個char字元。|
|gets(charArray)|遇到換行字元結束輸入，確認鍵留換行字元在緩衝區。|
|puts(charArray)|輸出字元陣列或字串並換行。|

## 算術運算子

- `a+b`
- `a-b`
- `a*b`
- `a/b`
- `a%b`取餘數

## 指定運算式

- `a += b`: 等同於 `a = a + b`.
- `a++`: 等同於 `a = a + 1`.
- `++a`: 等同於 `a = a + 1`.
- `a--`: 等同於 `a = a - 1`.
- `--a`: 等同於 `a = a - 1`.

## a++ vs. ++a

- 都是 `a = a + 1`.
- `a++`: 顯示或判斷後加 1.
- `++a`: 加 1 後顯示或判斷.
